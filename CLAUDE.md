## Fundamentals

- Use MVVM architecture.

## Api communication

- Use the autogenerated client through the openapi.json file.
- read the openapi.json file to understand the api structure.
- the client must to be used in the viewModel as a dependency.

## Before writing code

- Check with MCPs to see if you are using the correct implementation and the arguments are updated according with the version.
- Check if the property is only for mobile if that the case use the conditionals #if os(iOS) or #if os(macOS)
- Keep it simple and easy to understand.
